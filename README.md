# Full Stack Website

A modern, responsive PHP web application with authentication, notifications, and mobile-first design.

## Features

- User authentication (login/register) with secure forms
- Notification bell with dropdown alerts
- Responsive header and navigation menu
- Mobile-optimized authentication modal and forms
- Image lightbox for zooming images
- About and Contact pages
- Animated transitions and touch-friendly UI
- Modular PHP includes for header, footer, and modals

## Technologies Used

- PHP (server-side logic)
- JavaScript (interactive UI)
- CSS (responsive design)
- BoxIcons (icon library)

## Folder Structure

- /assets (images and style)
- /controllers(handle the login and the form submission logic)
- /includes (footer and the header)
- /pages (about and contact pages)
